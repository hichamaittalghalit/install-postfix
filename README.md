### INSTAALL `git`

<br>

    apt install git -y

<br>


### PULL THE PROJECT

<br>

    git clone https://github.com/hichamaittalghalit/install-postfix/

<br>

### PERMISSION

<br>

    cd install-postfix && sudo chmod 777 install.sh

<br>


### .ENV

<br>

    cp env.example .env

<br>

### LAUNCH THE SCRIPT

<br>

    sudo ./install.sh
